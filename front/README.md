# DentalScan

AI-powered assistant for dental CT analysis.

Product concept demonstrating how AI can support dentists in analyzing CBCT scans and identifying potential clinical findings.

## Problem

Dentists spend significant time analyzing CBCT scans due to:

- large number of slices
- complex anatomical structures
- risk of missing pathology

## Solution

DentalScan provides AI-assisted analysis of CT scans and highlights potential findings such as:

- periapical lesions
- bone loss
- impacted teeth

## Demo

Prototype includes:

- Dashboard with CT studies
- Upload CT flow
- AI analysis simulation
- Findings panel
- Clinical report generation

## Tech stack

- React
- Vite
- TailwindCSS
