# DentalScan

**DentalScan** — концепт AI-продукта для помощи стоматологам при анализе КТ/CBCT-исследований.

Проект демонстрирует, как AI-интерфейс может сократить время анализа снимков, подсвечивать потенциально значимые находки и помогать врачу быстрее формировать предварительный клинический отчёт.

## О проекте

DentalScan задуман как B2B SaaS-продукт для стоматологов и клиник.  
Цель - упростить работу с КТ-исследованиями, снизить нагрузку на врача при просмотре большого количества срезов и ускорить первичную интерпретацию данных.

В текущей demo-версии показаны ключевые пользовательские сценарии:

- dashboard со списком исследований
- загрузка нового исследования
- запуск AI-анализа
- просмотр исследования с AI-подсветкой находок
- панель findings
- черновик клинического отчёта

## Какую проблему решает

Анализ CBCT-исследований в стоматологии требует времени, внимания и высокой концентрации.  
Сложности, которые решает продукт:

- большое количество срезов
- риск пропустить патологию или анатомически значимую зону
- высокая когнитивная нагрузка на врача
- отсутствие быстрых вспомогательных инструментов для первичной интерпретации

## Решение

DentalScan предлагает AI-assisted workflow:

1. врач загружает КТ/CBCT-исследование
2. система выполняет предобработку данных
3. AI выделяет потенциально значимые области
4. врач просматривает findings
5. на основе findings формируется черновик отчёта для клинической верификации
